# Fake News Detection System

## Overview

This project is a Machine Learning-based Fake News Detection System that classifies news articles as Real News or Fake News using Natural Language Processing (NLP) techniques.

## Features

* Data Preprocessing and Cleaning
* Exploratory Data Analysis (EDA)
* TF-IDF Feature Extraction
* Logistic Regression Model
* Real-Time News Classification
* Confidence Score Display
* Streamlit Web Application

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-Learn
* TF-IDF Vectorizer
* Logistic Regression
* Streamlit

## Dataset

Kaggle Fake and Real News Dataset

## Model Performance

* Accuracy: 98.93%
* Precision: 98.66%
* Recall: 99.36%
* F1 Score: 99.01%

## Project Workflow

News Article → Text Cleaning → TF-IDF Vectorization → Logistic Regression → Prediction → Confidence Score
